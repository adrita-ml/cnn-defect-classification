CNN-Based Surface Defect Classification

📌 Overview
This project uses Convolutional Neural Networks (CNN) to classify surface defects in images. The goal is to automate quality inspection in manufacturing environments.

🚀 Problem Statement
Manual defect detection is time-consuming and error-prone. This project aims to build a deep learning model that can automatically classify defective vs non-defective surfaces.

🧠 Approach
- Image preprocessing and normalization
- Dataset filtering (defect vs non-defect classes)
- CNN model built using TensorFlow/Keras
- Model training and evaluation

📊 Results
- Achieved ~70% test accuracy
- Demonstrated capability of CNNs for visual inspection tasks

🛠️ Tech Stack
- Python
- TensorFlow / Keras
- NumPy

📈 Key Learnings
- Understanding CNN architecture
- Working with image datasets
- Model training and validation

▶️ How to Run
1. Install dependencies:
  pip install -r requirements.txt
2. Run the model:
  python src/cnn_model.py
